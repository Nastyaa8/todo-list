# Todo List

**Todo List** — это веб-приложение для управления задачами. Позволяет добавлять, удалять и отмечать выполненные задачи, а также фильтровать их по поиску. Проект написан на **HTML, CSS и Vanilla JavaScript**, без использования фреймворков!

##  Функционал

- Добавление новых задач
- Удаление отдельных задач и всех задач сразу
- Отметка выполненных задач с зачёркиванием
- Фильтрация задач по поиску
- Адаптивный дизайн для десктопа и мобильных устройств
- Сохранение задач в **LocalStorage**, чтобы не потерять данные

##  Технологии

- HTML5
- CSS3 / Flexbox / Grid
- JavaScript (ES6+)
- LocalStorage

##  Как запустить

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Nastyaa8/todo-list.git
```

2. Откройте index.html в браузере

3. Готово! Можно добавлять и удалять задачи
